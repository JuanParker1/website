Dark UI and Usability

More UI updates, I ran a lot of tests and tested some tooling against Bootstrap. In short Bootstrap is not yet ready for dark mode, the solution, I think is the best, is to add more colors, this time for backgrounds but they have so many developers and so many ideas they'll probably find some better way. My tools experimented with adjusting colors on the HSLuv scale and pretty much worked but currently Bootstrap allows placing primary color on primary background and does not see anything wrong with the fact that both are the same color. Hence, my idea of having a separate theme palette with background colors, but all components that have backgrounds (buttons, alerts, drop-downs) would have to be rewritten to make use of the new separate background palette... better than automatically calculating it during compilation. Then simple tools could be made to generate backgrounds from primary colors. Like the one I made: https://github.com/catpea/catkin.
