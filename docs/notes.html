Early Launch

Things maybe broken for a few days, but almost everything works https://news.catpea.com/ will be up and recent a while longer, it crashed Chrome on my cheap phone again, YouTube is still working great, but the poems may lag a bit. The best way to listen to latest poems is by going to https://catpea.com/ and clicking listen on the individual poems. I tested multiple audio players and they all had problems. YouTube prevents automating uploads, and all the audio platforms are making it difficult or impossible. I can't combine all the mp3 files into one, because git hub may start complaining about huge new files being added daily (there is no delete on git/github) it is OK to add one poem, bit not all poems in one new mp3 file every day. YouTube can work, I would need to upload a lot manually, but I would have two playlists, so things would be updated daily, and all poems would eventually catch up. I got to ship/deploy the new website today test it live, to start clearing up my to-do, simplify everything and begin automating as much as possible.
