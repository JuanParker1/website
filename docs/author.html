Cat Pea

Cat Pea, as all good things, is an Open Source Art Project. One of the key reasons behind it is the creation of my Art & Programming Portfolio.

The development of this project begun during 2020 Pandemic, and its minor goals are still very fluid.

It should run on a Raspberry PI Zero (including the build system), the graphic design should make it to BootstrapThemes/WrapBootstrap/CreativeMarket and make at least one sale. It should support many books, and help the authors with releasing Audio Book versions.

Cat Pea Industries hopes to inspire. Cat Pea is a study in Art and Typography, User Interface Design, User Experience Crafting, CSS, SASS, Bootstrap, git GitHub and GitLab, Debian and Linux From Scratch (LFS), Continuous Integration, Node.js, Node Modules, and Node Deployment, Electron Applications and Custom Code Editors. DNS Management, Raspberry PI Clusters, ZeroMQ Clusters, ImageMagick, GIMP, Audacity, ffmpeg, Sound and Signal Processing, YouTube Video Management, practicing UNIX, memorizing new commands and ideas, Static Site Generation and Resource Management, Custom static site generators and data queries. It is a challenge in Philosophy and Poetry, thinking on late evenings and early mornings during a 10K with heavy dumbbells.
